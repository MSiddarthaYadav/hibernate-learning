Welcome to my **Hibernate Learning Repository**! This repository showcases my learning journey through **Hibernate ORM** and **JPA**, covering everything from the fundamentals to advanced database mapping and querying concepts.

The repository contains practical examples, hands-on implementations, and well-structured code to build a strong foundation in Java persistence and database interaction.

---

# 📖 About

Hibernate is one of the most popular Object Relational Mapping (ORM) frameworks in Java. It simplifies database operations by mapping Java objects to relational database tables, reducing boilerplate JDBC code and improving application maintainability.

This repository documents my practice and learning of Hibernate concepts during my Java Full Stack learning journey.

# 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/MSiddarthaYadav/hibernate-learning.git
```

### 2. Open the project

Open the project in **VS Code** or **IntelliJ IDEA**.

### 3. Configure the database

Update the database connection details in the Hibernate/JPA configuration file.

### 4. Build the project

```bash
mvn clean install
```

### 5. Run the project

Execute the main application class to test Hibernate operations.
